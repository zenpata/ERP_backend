Global FE Behaviors — บังคับใช้ทุก sub-flow

  • ส่ง Authorization: Bearer <token> ทุก request
  • Response เป็น binary (application/pdf, application/zip, .xlsx) → ใช้ blob + URL.createObjectURL
  • Response เป็น JSON preview (invoices/:id/preview) → render HTML ใน modal
  • แสดง loading state บนปุ่ม (disabled + spinner) ระหว่างรอ
  • Success → toast ✅ พร้อมชื่อไฟล์
  • Error 401 → redirect login; 403 → toast "ไม่มีสิทธิ์"; 404 → toast "ไม่พบเอกสาร"; 5xx → toast + Retry
  • Async export: poll status endpoint → แสดง progress banner → เมื่อ done แสดงปุ่ม Download
